CRM for Economic Development

Beacon

A FOIA-aware CRM purpose-built for state economic development agencies. Three-zone architecture separates shared data, state-internal records subject to FOIA, and partner-private information.

Built for CT DECD and AdvanceCT.

Live Site GitHub

Document Signing

Manifest

Document signature workflows for grants and contracts. Route documents for review, collect e-signatures, and maintain audit trails. A standalone product that also integrates as a module of Harbor.

Legislative Intelligence

Lookout

AI-powered legislative tracking for CT state government. Automatically syncs bills from the General Assembly, scores relevance using Claude AI against configurable watch profiles, and manages the full testimony lifecycle from drafting through signature collection.

Live Site GitHub
Django 5.1 PostgreSQL Claude AI htmx Bootstrap 5 Open States API
FOIA Workflow

Admiralty

End-to-end FOIA request management for government agencies. Tracks requests from intake through response, catalogs statutory exemptions, and uses Claude AI to flag potential misclassifications during record review. A standalone product extracted from Beacon.

Django 5.2 PostgreSQL Claude AI htmx Bootstrap 5
